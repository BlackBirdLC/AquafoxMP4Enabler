# Aquafox MP4 Enabler

This is a plugin for Aquafox and all TenFourFox-based browsers to provide MP4 video playback support. Much like the other Aquafox applications, this is also an updated version of an application originally developed by [Cameron Kaiser](https://github.com/classilla).

This was supposed to launch alongside Aquafox 2.0, but sometimes life just gets in the way. FFmpeg 3.4.14 came out in early 2025 and is included in this repository, you can try building it yourself by simply running `gmake` in the `_mp4enabler` directory. See the Makefile for package requirements.
